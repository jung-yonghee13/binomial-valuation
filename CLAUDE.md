# CLAUDE.md

## 하네스: 가치평가 앱 닥터 (진단·수정·디자인)

**목표:** 이항모형 가치평가 웹앱(binomial-valuation, Streamlit Cloud)의 에러·가치평가 실패를 여러 계층에서 진단·근본 수정·검증·배포·문서화하고, 대시보드 UI를 Valuation Suite 디자인으로 구축·조정한다.

**트리거:** 웹앱 에러·"평가 실패"·가치평가 이상, 또는 대시보드 디자인/레이아웃 변경 요청 시 `valuation-app-doctor` 스킬(오케스트레이터)을 사용하라. 후속 요청("다시 진단", "이 부분만", "재배포")도 같은 스킬로 이어간다. 단순 개념 질문은 직접 응답 가능.

**실행 참고:** 팀 실행 도구(TeamCreate/SendMessage/Task*)는 harness 플러그인 소속이라 `/reload-plugins` 후 활성화된다. 미활성 시 오케스트레이터가 서브 에이전트(`Agent`, `model:"opus"`)로 대체 실행한다. 에이전트 정의는 `.claude/agents/`, 스킬은 `.claude/skills/`.

**핵심 원칙:** "조용히 틀린 값 금지" — 폴백/대체 데이터는 결과·보고서에 근거와 함께 투명하게 노출. 외부 반영(push·Notion)은 사용자 승인 후.
